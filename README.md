# OLABS

A modern, scalable web application built with React and TypeScript. 

## 🚀 Tech Stack

This project leverages a robust modern frontend stack:

* **Framework:** React 18 with TypeScript.
* **Build Tool:** Vite for fast, optimized development and bundling.
* **Styling:** Tailwind CSS combined with Shadcn UI (`@radix-ui`) for accessible, customizable components.
* **Routing:** React Router DOM for seamless single-page application navigation.
* **State Management & Data:** TanStack React Query (`@tanstack/react-query`) for powerful asynchronous state management.
* **Backend & Auth:** Integrated with Supabase (`@supabase/supabase-js`).
* **Testing:** Vitest for unit testing and Playwright for end-to-end testing.
